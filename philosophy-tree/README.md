# Tree Hotspots Prototype

Простой одностраничный прототип: картинка + SVG-hotspots + подсветка + описание по hover/tap.

## Как открыть

Открой `index.html` в браузере.

## Файлы

- `index.html` — разметка и SVG-hotspots
- `style.css` — внешний вид, подсветка, карточка описания
- `script.js` — hover/tap/focus логика
- `assets/tree.png` — изображение дерева

## Режим координат

Открой:

```text
index.html?coords=1
```

Наводи мышь или тапай по картинке — внизу слева появятся координаты `x/y` в системе SVG `0..100`.

## Как править зоны

В `index.html` найди SVG-блок и меняй координаты у фигур:

- `rect` — земля
- `ellipse` — семя и плоды
- `path` — корни, ствол, ветви, листья, кислород, семена

SVG использует:

```html
viewBox="0 0 100 100"
```

Поэтому координаты удобно править вручную.
